# webflux-mongodb
Проект содержит пример микросервиса, реализованного на WebFlux.
Приложение принимает, сохраняет чеки ОФД и представлет сохраненные данные. 
Данные принимаются в формате XML, конвертируются в формат Bson Document и сохраняются в базе данных mongodb.
Взаимодействия с базой данных реализовано непосредственно через com.mongodb.reactivestreams.client, 
без использования Spring repository. 
Контроллер реализован традиционно(не в функциональном стиле), с испрользованием аннотаций Spring MVC.

Пример обрабатываемых данных в файле test/data/check.xml.
